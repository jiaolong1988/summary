ctrl+alt系列
	Ctrl+Alt+O              	清除无效import导入包
	Ctrl+Alt+M					快速抽取方法					
	Ctrl+Alt+T                  Surround with 快捷键,try/catch快捷键				
	Ctrl+Alt+Enter			    切换到上一行代码空位
	Ctrl+Alt+方向键(下)			复制当前行内容至下一行
	Ctrl+Alt+H					查看方法被哪里引用
	Ctrl+Alt+B					查看方法在哪里定义	
	Alt+Enter					获取对象的所有get set方法


万能键
	Ctrl + 1					帮助,自动生成变量名
	Ctrl+Alt+O                  清除无效import导入包
	Alt  + F					控制台搜索

代码辅助与移动
	Ctrl + C					复制整行代码
	Ctrl+shift +F   			格式化代码
	Tab 						代码补全

	shift+Enter					切换到下一行代码空位 
	Ctrl+Alt+Enter				切换到上一行代码空位
	
	Alt + 方向键(左或右)    	返回上一次操作位置
	Alt + 方向键(上或下)   		移动代码至上一行或下一行
	
	Ctrl+Alt+方向键(下)			复制当前行内容至下一行
	Ctrl + D  					删除当前行
	
	Ctrl+Alt+方向键(右)			类文件之间切换[下一个tab类窗口]
	Ctrl+Alt+方向键(左) 		类文件之间切换[上一个tab类窗口]


结构查找
	Ctrl+Alt+H					查看方法被哪里引用
	Ctrl+Alt+B					查看方法在哪里定义				 
	Ctrl + T 					接口的实现类有哪些。选中接口方法，直接跳转到实现类。

文件查找
	Ctrl + F					在当前文件中 查找与替换
	Ctrl + H					查询包含关键字的文本文件(根据内容查找文件)
	shift+shift					全局查找文件

类所有方法查看
	Alt  + 7					查看类中的所有方法 
	Ctrl + O					查看所有方法 包含继承类的方法
	Ctrl + E 					显示最近编辑的文件列表

代码生成
	String.list					快速创建：List<String> list =new ArrayList<>();
	String.map					生成创建：Map<String, String> map =new HashMap<>();
	 
	Alt + I    				    生成构造方法
	Ctrl+Alt+M					快速抽取方法						
								选中指定代码 -> 右键->Refactor -> Extract Method
	Ctrl+shift U				大小写转换
	Ctrl+Alt+T                  Surround with 快捷键,try/catch快捷键								

debug
	Ctrl + R					debug代码到光标处（2021版本）
			
	
注释
	/**							类注释的 快捷方式
	/*							方法的   快捷方式
	/							纯粹注释 的快捷方式
	Ctrl+shift+/				对选中代码进行注释/*  */


代码展开与折叠
    Ctrl + =					Expand All   (展开全部方法)
	Ctrl + -					Collapse All (折叠全部方法)

复制类全路径
	Alt + C						复制类的全路径,包名+类名
	Alt + V						粘贴类的全路径
	

	
查看类的继承关系图	
	右键类 -> Diagrams -> Show Diagram Popup
	Crtl + Alt + U									临时展示
	Crtl + Alt + Shift + U       					永久展示

TranslationPlugin 翻译的快捷键
	显示翻译对话框 		Ctrl + Shift + O
	翻译           		Ctrl + Shift + Y
	翻译并替换			Ctrl + Shift + X			将指定中文替换为英文
