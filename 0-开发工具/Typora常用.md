

# 快捷键

| 快捷键         | 功能                                                         |
| -------------- | ------------------------------------------------------------ |
| Shift + Tab    | **减少缩进**                                                 |
| Ctrl + T       | 插入表格                                                     |
| == 文字内容 == | 高亮:typora 文件->票号设置->Markdown->Markdown扩展语法，选中==高亮== |
| enter enter    | 结束序列                                                     |
| enter tab      | 子序列                                                       |
| :符号名称      | 输入特定符号。例如（:one）:one:                              |
|                |                                                              |
|                |                                                              |
